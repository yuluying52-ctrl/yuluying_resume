# 于璐滢 · 个人简历（静态站）

部署在 **GitHub Pages** 后访问：

**https://yuluying52-ctrl.github.io/yuluying_resume/**

（需先在 GitHub 仓库 **Settings → Pages** 中开启：`main` 分支、`/` 根目录。）

## 更新网站

在项目文件夹执行：

```bash
git add .
git commit -m "更新"
git push
```

## 推送若提示 403 / denied to yuluying

说明当前登录的是另一个 GitHub 账号。请用 **`yuluying52-ctrl`** 重新登录后再 `git push`，或见仓库内 `推送说明.txt`。
